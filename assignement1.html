Dear All,

Welcome to my first HTML page with CSS design. Here I would like to include the thinking process while designing this page.

First of all, I consider this website as my first personal website, which could be used to introduce who I am during and after my one-year postgraduate study. Therefore, I put my name on the top line of the page. Following this idea, the function of “About Me” and “Contact Me” are added as a sidebar next to the heading. As for the “About Me” function, it is linked to my LinkedIn page, so that I do not need to cover all the information once again and people can easily build connections with me via this popular social platform. In order to raise people’s attention, I applied to hover function on the sidebar to highlight these functions as well. Regarding “Contact Me”, I have not included an external link since I do not have other communication accounts that are open to the public besides LinkedIn so far. The second part, the catalog, is essential because it is the required function in our first assignment. In the first HTML draft, the content of the catalog was considered as the main body of the page, and the main body, which is this article, was missing. However, I felt like it would be a little boring and empty if finalising the page as that. Therefore this article reflecting my design thinking process is added! Going back to the catalog section, only “assignment1” have the hover function and URL because other assignments have not been introduced so far. The last function is the footer. This is included because I think people who are interested in my experience, as well as the skills used to design this page, would like to check out which uni I went to and which courses I have taken. So, do you think my website is interesting that YOU would also like to learn some basic HTML/CSS to make your own websites? If so, please check it out in the footer!

Apart from functions, the design aesthetic is also worth mentioning. I firstly used a picture taken by myself as the background picture, and then adjusted the background clolour, colours of the text, and font family such as “fantasy”. By the way, font-style "fantasy" appears in different ways via google chrome and safari, which is quite interestig. However, this kind of design is a little bit “old school” that does not match trendy website aesthetics nowadays. Keeping with the idea “less is more”, I deleted that background image and put it in the main body as a supporting image of this article. I still keep the background colour to avoid being too boring and the colour also matches the color of the image. Meanwhile, I use the flat font family as it is closer to the “modern” aesthetics of website design. Now it looks like a modern newspaper page! So, that’s all about this page, my first HTML/CSS design. Isn’t that interesting? If you like it or have some advice, do hesitate to leave me a message or DM on LinkedIn.

Thanks and goodbye.

Best wishes,
